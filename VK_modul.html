ℹ️ Základní příkazy a zápis
Význam základních příkazů
• G111 = Definice pólu (VPOL) pro určení úhlů a konstrukčních drah.
• G11 = Volná úsečka bez rozměrů (X?, Z?) nebo polárně přes PA/PR.
• G2 / G3 = Volný oblouk s poloměrem R.
Možnosti zápisu geometrie
• Průsečíky (?): Neznámé se ručně přepíší po externím výpočtu.
• Tečnost (T): Plynulé napojení na předchozí prvek.
• Konverze: Po doplnění hodnot lze kód převést na čistý standardní ISO G-kód.
Formáty pro uložení (Export)
• .NC / .TXT = Univerzální textový ISO G-kód.
• .DIN = Standard dle normy DIN 66025.
• .MPF = Hlavní program kontury (Siemens).
• .SPF = Technologický podprogram.
🔍 Význam parametru VPOL
Kdy je VPOL nutný?
Při křížení prvků se dvěma matematickými řešeními. Určuje jednoznačnou dráhu:
• VPOL1 = První řešení (bližší bod od počátku / spodní průsečík).
• VPOL2 = Druhé řešení (vzdálenější / horní průsečík).
